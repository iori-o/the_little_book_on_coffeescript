[nodebrew をインストールする方法](http://qiita.com/items/9236fd6409eb564abd71)

## CoffeeScript のインストール

``` bash
% npm install -g coffee-script
```

## 実行方法

* コマンドラインオプションを与えずに実行すると CoffeeScript のコンソールに入る。

* ファイルのコンパイルを行うには `--compile` オプションを与える。

* ファイルのデフォルト拡張子は `.coffee` になる。
